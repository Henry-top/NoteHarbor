# 测试说明

完整检查：

```bash
pnpm lint
pnpm test
pnpm build
cargo test --manifest-path src-tauri/Cargo.toml
```

自动测试覆盖：

- YAML 解析与旧 Markdown 不改写
- 标签、字数、大纲和双向链接转换
- 安全路径与非法文件名
- Markdown 索引、读取和中文标签
- 改名后的双向链接和相对链接更新
- 10,000 篇笔记的 SQLite FTS5 搜索响应

发布前手动验收：

1. 同时添加两个资料库并完成索引。
2. 创建普通笔记和每日笔记。
3. 在三种编辑模式间切换并检查公式、Mermaid、表格和任务列表。
4. 粘贴图片，确认进入资料库根目录的 `assets/`。
5. 添加标签、收藏、置顶和双向链接。
6. 重命名被引用笔记并确认链接自动更新。
7. 使用全局搜索打开另一资料库中的笔记。
8. 用其他编辑器修改已打开的文件，确认出现冲突保护。
9. 等待历史快照生成，重启应用后恢复旧版本。
10. 切换三套主题及明亮、深色、跟随系统模式。
